# PaSST

This directory contains PaSST-related model code used as a backbone by this
research project. The code was obtained from the public PaSST repository and
includes code adapted from the timm vision-transformer implementation, as noted
in the source files. It may also include local integration changes.

- Upstream project: [PaSST](https://github.com/kkoutini/PaSST)
- Related upstream project: [pytorch-image-models (timm)](https://github.com/huggingface/pytorch-image-models)
- Upstream PaSST license: Apache License 2.0
- License copy: see the repository-root [LICENSE](../../LICENSE)

The upstream projects are the authoritative sources for their documentation,
pretrained-model information, downloads, and updates. This repository is not
the official PaSST or timm distribution and is not affiliated with or endorsed
by their maintainers.

Pretrained checkpoints are not redistributed here. Download required checkpoints
from the upstream project and keep them outside version control.
