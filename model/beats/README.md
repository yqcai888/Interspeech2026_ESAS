# BEATs

This directory contains BEATs model code used as a backbone by this research
project. The code was obtained from Microsoft’s public UniLM repository and may
include local integration changes.

- Upstream project: [Microsoft UniLM — BEATs](https://github.com/microsoft/unilm/tree/master/beats)
- Upstream copyright: Copyright (c) Microsoft Corporation
- Upstream license: MIT; see [LICENSE](LICENSE)

The upstream project is the authoritative source for its documentation,
pretrained-model information, checkpoint downloads, and updates. This repository
is not the official BEATs distribution and is not affiliated with or endorsed by
Microsoft.

Pretrained checkpoints are not redistributed here. Download required checkpoints
from the upstream project and keep them outside version control.
